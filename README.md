# Xgui4 OS Dotfiles

My dotfiles of my GNU/Linux System, including HyprArch (my hyprland setup) and i3wm and qtile config and terminal config and a work in progress installer

## Hierarchy

- hypr : Hyprland confgi
- polybar : polybar for i3wm
- picom : Picom for X Window Manager
- i3 : i3wm with XLibre
- rofi : rofi menu configs
- system : system file to modifed in root (require root)
- home-folder : config file not located in .config/ folder
- qtile : qtile config
- qtxct : config for Qt 5 and 6
- gtk-x : config for GTK 3 and GTK 4
- waybar : config for Waybar
- wallpapers : wallaper using by the dotfiles
- [without a folder] : files in .config/ or used for the scripts/installer
- picom : X Compositor for eye candy for XLibre Window Manager
- nwglook : config for GTK App
- fish and kitty and starship.toml : termianl and shell config
- fastfetch : fastfetch config
